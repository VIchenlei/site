精准定位设备

1、读卡分站

读卡分站是精确定位管理系统的主要设备。分站不断采集其信号范围内的车辆、人员标识卡信息，并按照TCP/IP协议，通过光缆或电缆将接收到的标识卡信息上传给控制中心计算机；同时，分站接收控制中心计算机下发的各种指令，并做出相应的操作。

产品特点：

  • 1. 具有与交换机双向通信功能及工作状态指示功能；
  • 2. 具有无线编码收发功能，可与车辆标识卡双向通信；
  • 3. 具有掉电保护功能；
  • 4. 具有自诊断和故障指示功能；
  • 5. 具有通讯信号中断后的数据存储和续传功能；
图1 读卡分站

针对不同的应用场景，北交智慧开发了不同的读卡分站。在有些矿井环境中不具备光纤通信，北交智慧研发了只通过 CAN 进行通信的微型读卡分站。

图2 微型读卡分站

2、车辆标识卡

车辆标识卡可安装在车辆或其他移动目标之上，具有无线通信功能，可与读卡分站进行双向通信，并将信息发送给分站。

产品特点：

  • 1. 标识卡具有无线编码收发功能；
  • 2. 标识卡具有电池欠压提示功能；
图3 车辆标识卡

3、人员标识卡

人员标识卡佩戴方便，具有无线通信功能，可与读卡分站进行双向通信，并将信息发送给分站。标识卡可通过无线通信进行呼叫，并可在被呼叫时作出响应。

产品特点：

  • 1. 具有无线编码收发功能；
  • 2. 具有电池欠压提示功；
  • 3. 具有报警信息指示功能；
  • 4. 具有按键呼叫功能；
图4 人员标识卡
